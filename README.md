## About Me

I'm **YEMENI**, a RE4 modder and tool developer from Yemen.

I specialize in **Resident Evil 4 Ultimate HD Edition (2014)** — working deep with its binary file formats, building editors and tools, and creating complex mods from scratch.

I build all my tools with the help of **Claude (Anthropic AI)** as my primary implementation partner, while handling all testing, debugging, integration, and design decisions myself.

I work entirely on mobile — **Honor X8 5G** — using **Winlator** to run Windows tools and compile projects.

- RE4 modder active in the Arabic-speaking community
- Tool developer with hands-on expertise in RE4 file formats: AEV, ITA, ESL, OSD, and more
- Tutorial creator — all tutorials written in Arabic
- From Yemen 🇾🇪
